Example Ping! Simple

This is the most simple example for Journey Builder Custom Activity. This activity sends a message to the service endpoint and replies back with a message you specify in the activities settings.

Add: {{domain+path}}/ping/ to Marketing Cloud -> Setup -> Platform Tools -> Apps > Installed Packages

The Source Code for UI is located at src/modules/ping and the source code for the API is located at controllers/pingController.js.
